Приложение управления доставками

Клиент

ООО “Дайрекст” (Direxxt) - компания, которая оказывает услуги по экспресс доставке отправлений, в том числе к определенному времени, транспортные услуги, курьерские услуги, транспортно-экспедиционные услуги.

Проблема

  • Компания ООО Дайрекст хотела сократить расходы по доставке грузов, и отказаться от услуг системы «БТС», благодаря собственной системе построения маршрута, управления доставками, увеличить качество и скорость управления доставками.

Решение

Для пользователей, которым нужны услуги по прямой экспресс доставке по Беларуси, было разработано клиентское приложение, где клиент может сам создавать заказы и видеть их статусы.

Раздел администратора позволяет создавать оптимизированные маршруты по доставке груза и отслеживать данные по автомобилю и водителю, чтобы иметь возможность оптимизировать распределение заказов. Здесь есть блоки с расписанием водителей, каталог транспорта, список сотрудников, список клиентов, список заказов и самый главный – Карта и Маршруты.

Водительское приложение позволяет водителям принимать заказы, менять их статус, сообщать о проблемах с заказами.
